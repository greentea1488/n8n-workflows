# Budget Weekly Reporter — отчёт по Google Sheets

## Описание

Автоматический недельный отчёт по бюджету. Workflow по расписанию читает данные из Google Sheets, считает расходы за прошлую неделю, группирует по категориям и формирует человекочитаемый текстовый отчёт.

Можно использовать как:

- личного финансового ассистента,
- семейный бюджет‑репорт,
- основу для дашбордов.

---

## Используемые интеграции

- **Schedule Trigger** — запуск по расписанию (например, раз в неделю)
- **Google Sheets** — источник данных (таблица с тратами)
- **OpenAI Chat** — генерация итогового текста отчёта

---

## Как запустить

1. Импортируйте `workflow.json` в n8n.
2. Пропишите креды Google и OpenAI.
3. Укажите:
   - ID вашей таблицы,
   - название листа с тратами.
4. Настройте расписание (раздел Schedule).
5. Активируйте workflow — по расписанию он будет собирать данные за прошлую календарную неделю и формировать текстовый отчёт.
