# neuroCRM Legal PDF Automation — Bitrix24 → PDF претензии

## Описание

Продвинутый enterprise‑workflow для автоматизации юридических претензий по авторскому праву. Он:

- принимает вебхуки от **Bitrix24** при создании сделки,
- проверяет токен и стадию сделки,
- вытаскивает данные по компании и нарушению,
- формирует текст претензии (с отсылкой к нормам ГК РФ),
- отправляет данные в сервис генерации PDF (например, APITemplate),
- получает готовый PDF‑файл, который можно отправить по e‑mail.

---

## Используемые интеграции

- **Bitrix24 (Webhook + REST)** — источник данных о сделках
- **Code/Function Nodes** — генерация текста письма и подготовка данных
- **APITemplate.io или аналогичный сервис** — создание PDF на основе шаблона
- (Опционально) **SMTP / Gmail** — отправка письма с вложением

---

## Как запустить

1. Импортируйте `workflow.json` в n8n.
2. Настройте:
   - входящий вебхук из Bitrix24,
   - домен и токен Bitrix24,
   - сервис генерации PDF (URL, API key, ID шаблона).
3. Адаптируйте поля под вашу CRM‑схему (IDs стадий, кастомные поля).
4. Создайте тестовую сделку в Bitrix24 и убедитесь, что:
   - workflow подтягивает данные,
   - формирует корректный текст претензии,
   - генерирует и сохраняет PDF.
