# Hook Generator Client — маркетинговые хуки с LLM + MCP

## Описание

Ассистент, который генерирует цепляющие маркетинговые хуки и креативы для:

- постов,
- Reels и сторис,
- лендингов и e‑mail‑рассылок.

Использует LLM, память и (опционально) MCP‑инструменты для доступа к базе знаний о ЦА, продуктах и инсайтах.

---

## Используемые интеграции

- **Chat Trigger / Webhook** — входные запросы из UI/форм
- **AI Agent** — LLM‑агент с system prompt под генерацию хуков
- **OpenAI Chat** — модель для генерации текста
- **Simple Memory** — краткосрочная память
- **MCP Client** — доступ к внешним MCP‑инструментам (по желанию)

---

## Как запустить

1. Импортируйте `workflow.json` в n8n.
2. Настройте OpenAI API‑ключ.
3. (Опционально) Настройте MCP клиент (endpoint, токены).
4. Отправьте ассистенту описание продукта / оффера / боли ЦА — в ответ получите несколько вариантов хуков.
