# BudgetBot — Telegram + Voice + Google Sheets + Gmail

## Описание

Бот‑ассистент для личных финансов. Принимает:

- обычные текстовые сообщения,
- голосовые сообщения (распознаёт их через OpenAI),

и на основе этого помогает управлять бюджетом, анализировать траты и формировать ответы, опираясь на данные в Google Sheets. Может отправлять сводки и отчёты по e‑mail.

---

## Используемые интеграции

- **Telegram** — интерфейс общения с пользователем
- **OpenAI (Whisper + Chat)** — распознавание голоса и генерация ответа
- **Google Sheets** — хранение данных бюджета
- **Gmail** — отправка писем с отчётами

---

## Как запустить

1. Импортируйте `workflow.json` в n8n.
2. Настройте креды:
   - Telegram (бот‑токен),
   - Google Sheets (OAuth2),
   - Gmail (OAuth2),
   - OpenAI API‑ключ.
3. Укажите ID своей таблицы бюджета и (опционально) e‑mail для отчётов.
4. Активируйте workflow и напишите боту:
   - текстом или голосом — бот должен принять запрос, расшифровать голос и обработать его как текстовый ввод.
