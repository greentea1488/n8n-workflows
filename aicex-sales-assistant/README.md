# AICEX Sales Assistant — продающий Telegram‑бот

## Описание

Продающий Telegram‑ассистент с памятью, который:

- отвечает на вопросы клиентов,
- объясняет пользу продуктов (курсы, спринты, клуб),
- помогает с выбором,
- мягко подводит к покупке.

Бот построен на LLM и контекстной памяти (несколько последних сообщений), поэтому ведёт диалог как живой менеджер.

---

## Используемые интеграции

- **Telegram** — канал общения
- **OpenAI Chat** — LLM‑ядро для диалогов
- **Simple Memory** — краткосрочная память переписки

---

## Как запустить

1. Импортируйте `workflow.json` в n8n.
2. Пропишите токен вашего Telegram‑бота.
3. Задайте свой system prompt (описание продуктов, офферов, стиля общения).
4. Активируйте workflow и протестируйте: спросите бота о продукте — он должен рассказать о нём и подвести к регистрации/оплате.
